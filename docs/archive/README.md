# 归档说明

本目录保存 2026-05-28 文档瘦身时从根目录移出的材料。归档不是删除；它用于保留历史证据链，同时让根目录只承担当前执行入口。

## 目录

- `full_snapshots_20260528/root/`：整理前根目录 Markdown/JSON/DOCX 原样快照。
- `method_rounds/`：方法轮次方案、源码审计、赛马协议、验证深度复审。
- `planning_and_submission/`：投稿规划、领域路线、早期 CS-PANDA/PRCV 文档、阅读笔记。
- `registries/`：候选登记表。
- `word_docs/`：Word 文档和备份。

## 使用原则

- 查“现在做什么”：看根目录 `current_status.md` 和 `todo.md`。
- 查“为什么某个方法被降级”：看 `method_rounds/` 与 `registries/`。
- 查“早期投稿和选题判断”：看 `planning_and_submission/`。
- 查“整理前原文”：看 `full_snapshots_20260528/root/`。
